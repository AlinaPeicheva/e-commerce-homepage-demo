# bug club — e-commerce homepage

An editorial, high-end homepage concept for **Bug Club**, a tote bag brand. Built live for the SoftUni Creative seminar **"Новият дизайн процес с Figma MCP и Claude Code"**.

## About this project

This repository was created during the seminar to demonstrate an AI-assisted design-to-code workflow using [Claude Code](https://claude.com/claude-code):

- **[Higgsfield MCP](https://higgsfield.ai/)** was used to generate all product, lifestyle, and decorative imagery (tote bag product shots, the editorial hero photo, the flat-lay detail shot, and the illustrated field-guide graphic).
- **Figma MCP** was used to export the finished HTML/CSS page into Figma for design handoff.

### The prompt

> Help me create a fun and creative HTML e-commerce homepage for a tote bag brand "bug club". I've uploaded the logo and product photos as style references. Use the Higgsfield MCP to generate product, lifestyle or decorative images needed.
>
> The design should feel refined and modern—soft pastel tones with vibrant color accents, expressive typography, not a generic Shopify template. Avoid cookie-cutter sections, flashy gradients, or overly symmetrical layouts. I want an editorial, high-end look with thoughtful typography, subtle interactions, and distinctive details.
